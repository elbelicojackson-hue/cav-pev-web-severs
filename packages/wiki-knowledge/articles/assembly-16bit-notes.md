# 16位汇编笔记整理

## 寄存器总结

### 8086CPU 14个寄存器

| 类别 | 寄存器 | 用途 |
|------|--------|------|
| 通用寄存器 | AX (AH/AL) | 临时存储，参与乘除法运算 |
| 通用寄存器 | BX (BH/BL) | 存储偏移地址访问内存 |
| 通用寄存器 | CX (CH/CL) | 条件转移/循环计数 |
| 通用寄存器 | DX (DH/DL) | 辅助AX，参与乘除法 |
| 指针寄存器 | SI, DI | 存储偏移地址（不能拆分8位） |
| 指针寄存器 | SP | 栈偏移地址，SS:SP指向栈顶 |
| 指针寄存器 | BP | 基址指针，默认段在SS中 |
| 段寄存器 | CS | 代码段，CS:IP指向当前指令 |
| 段寄存器 | DS | 数据段，存放数据段地址 |
| 段寄存器 | SS | 栈段，存放栈段地址 |
| 段寄存器 | ES | 附加段 |
| 指令指针 | IP | 当前指令偏移地址 |
| 标志寄存器 | PSW | 存储指令执行结果/标志 |

### 寻址规则

- 只有 BX, SI, DI, BP 可以进行内存寻址
- BP 默认段寄存器为 SS，其他默认 DS
- 不能直接将数据送入段寄存器

## 汇编指令分类

### 数据传送指令

| 指令 | 功能 | 注意事项 |
|------|------|----------|
| MOV | 数据传送 | 不能两个内存单元直接传送，不能直接送CS，不影响标志位 |
| MOVSX | 带符号扩展传送 | 符号位填充高位 |
| MOVZX | 带零扩展传送 | 0填充高位 |
| LEA | 取地址 | lea取地址，mov取值；对寄存器加[]取值 |
| XCHG | 交换 | 寄存器/内存与寄存器交换 |
| PUSH/POP | 压栈/出栈 | 16位下操作16位数据 |

### 算术指令

| 指令 | 功能 | 说明 |
|------|------|------|
| INC/DEC | 加1/减1 | 不影响进位标志 |
| ADD/ADC | 加法/带进位加法 | ADC = 操作数和 + CF |
| SUB/SBB | 减法/带借位减法 | SBB = 操作数差 - CF |
| MUL | 无符号乘法 | 结果在DX:AX |
| IMUL | 有符号乘法 | 支持多操作数形式 |
| DIV/IDIV | 无符号/有符号除法 | DX:AX / 操作数 |
| NEG | 取反 | 符号取反 |

### 逻辑指令

| 指令 | 功能 |
|------|------|
| AND | 按位与 |
| OR | 按位或 |
| XOR | 按位异或（相同为0，不同为1） |
| NOT | 按位取反 |
| TEST | 逻辑与测试（不保存结果，只影响标志） |

### 串操作指令

| 指令 | 功能 | 类比 |
|------|------|------|
| MOVSB/W/D | 串传送 | memcpy |
| LODSB/W/D | 串加载 | 求和 |
| STOSB/W/D | 串送存 | memset |
| CMPSB/W/D | 串比较 | memcmp |
| SCASB/W/D | 串扫描 | strchr |

- 寻址：DS:ESI（源），ES:EDI（目的）
- 执行后自动修改ESI/EDI，方向由DF标志决定
- STD设置DF=1（逆向），CLD清除DF=0（正向）

### 重复前缀

| 前缀 | 条件 |
|------|------|
| REP | ECX!=0 时重复（配合MOV/LOD/STO） |
| REPE/REPZ | ECX!=0 && ZF==1 时重复（配合CMP/SCA） |
| REPNE/REPNZ | ECX!=0 && ZF==0 时重复（配合CMP/SCA） |

### 比较与跳转

| 指令 | 条件 |
|------|------|
| JMP | 无条件跳转 |
| JE/JZ | ZF=1（相等/零） |
| JNE/JNZ | ZF=0（不等/非零） |
| JB/JNAE | CF=1（无符号小于） |
| JNB/JAE | CF=0（无符号大于等于） |
| JL/JNGE | SF!=OF（有符号小于） |
| JNL/JGE | SF==OF（有符号大于等于） |
| JLE/JNG | ZF=1或SF!=OF（有符号小于等于） |
| JNLE/JG | ZF=0且SF==OF（有符号大于） |

### 循环与调用

| 指令 | 功能 |
|------|------|
| CALL | 调用子程序，压入返回地址 |
| RET/RETN | 返回，弹出返回地址跳转 |
| LOOP | ECX减1，非零则跳转 |
| LOOPZ/LOOPE | ECX减1，ZF=1且ECX!=0则跳转 |
| LOOPNZ/LOOPNE | ECX减1，ZF=0且ECX!=0则跳转 |

## 寻址模式

| 模式 | 公式 | 示例 |
|------|------|------|
| 立即寻址 | MOV EAX, 常量 | `mov ax, 0x1234` |
| 直接寻址 | MOV EAX, [常量] | `mov ax, [0x1234]` |
| 寄存器寻址 | MOV EAX, 寄存器 | `mov cx, bx` |
| 寄存器间接 | MOV EAX, [寄存器] | `mov ebx, [ebx]` |
| 寄存器相对 | MOV EAX, [寄存器+偏移] | `mov edx, [ebx+0x12]` |
| 基址变址 | MOV EAX, [基址+变址] | `mov edx, [ebx+esi]` |
| 相对基址变址 | MOV EAX, [基址+变址+偏移] | `mov eax, [ebx+esi+0x8]` |
